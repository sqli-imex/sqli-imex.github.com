SQLI Im/Ex

Exporting and importing Liferay company-level data

What ?

Liferay Portal provides a smart way to export and import site-level data, through the LAR files, but in certain scenarios it is also necessary to export and import some company-level data (for examples default permissions definitions for roles). With this tool, you will be able to achieve company-level data exporting and importing.
